    JavaScript Lenguaje de programación utilizado tanto en cliente como en servidor que permite hacer páginas interactivas https://www.javascript.com/
    TypeScript Lenguaje de programación desarrollado por Microsoft para aplicaciones de Javascript que se ejecutan tanto en cliente como en servidor https://www.typescriptlang.org/

    Prototype Framework en JavaScript como parte del soporte de Ajax para Ruby on Rails http://prototypejs.org/
    JSP Colección de tecnologías para crear páginas web dinámicas basadas en HTML, XML, SOAP u otros tipos de documentos https://en.wikipedia.org/wiki/Jakarta_Server_Pages
    Vue Framework para JavaScript utilizado para crear interfaces web y aplicaciones de una sola página https://vuejs.org/
    Angular Plataforma y framework para construir aplicaciones de cliente de una sola página utilizando HTML y TypeScript https://angular.io/
    DJango Framework en Python para el diseño rápido y limpio de aplicaciones web https://www.djangoproject.com/

    Gatsby Generador de webs estáticas https://www.gatsbyjs.com/
